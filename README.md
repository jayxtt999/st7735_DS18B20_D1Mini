# st7735_DS18B20_D1Mini
## Wemos D1 mini  0.96TFT(ST7735s)  DS18B20 显示温度

### 元器件
- D1mini（基于esp8266的都可以，注意引脚映射关系）
- DS18B20
- 0.96寸LCD显示屏（ST7735s）
